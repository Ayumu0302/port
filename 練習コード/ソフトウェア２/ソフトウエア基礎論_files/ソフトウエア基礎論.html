htmlとcssレイアウト

htmlでもワープロのように中央揃え、右揃えを始め、文章、画像などのレイアウトを設定することができます。厳密には、html自体は、文書、画像などのコンテンツを示すためにタグを用い、デザイン、レイアウトにはCSSを用いるという考え方がベースになっています。CSSを駆使することで、ワープロと同様に様々な文書デザインが可能です。

htmlのレイアウトの基本はボックスレイアウトです。これまで紹介したh2タグ、pタグなどには、タグ間の余白（Margin)、境界線（Border)、境界線と内容の間(padding)の大きさを設定することができ、一つのボックスと考えることができます。 そして、それぞれのボックスをどのように配置するかを決める方法も複数あり、検索すると様々な方法を 見ることができます。

inline-blockで横並び

ここでは、横並びにする方法を紹介します。横並びにする方法だけでも複数の方法がありますが、 この授業では、比較的簡単なinline-blockを用いる方法を推奨します。

pタグは段落という意味がありますが、レイアウト上、複数の段落と写真をひとまとまりにするなど、 文章として意味がないものをボックスとしたい場合があります。その場合に用いるのがdiv(division)タグです。

実例で紹介したいと思います。 htmlファイルのbody内に下記コードを入力しましょう。 

<div class="width1000">
    <img src="keyboard.jpg" alt="keyboard">
    <img src="keyboard.jpg" alt="keyboard">
    <img src="keyboard.jpg" alt="keyboard">
 </div>
<div class="width100">
    <img class="width25" src="keyboard.jpg" alt="keyboard">
    <img class="width25" src="keyboard.jpg" alt="keyboard">
    <img class="width25" src="keyboard.jpg" alt="keyboard">
    <img class="width25" src="keyboard.jpg" alt="keyboard">
</div>

さらに、下記cssを読み込むようにしてください。 

img{
  display: inline-block;  /* インラインブロックに */
  /*border: 1px red solid;*/  /* 枠線を太さ1px、色を赤、普通の線 (確認用)*/
}
.width25{
  width: 25%;
}
.width100{
  width: 100%;
  text-align: center;     /* 中身を中央ぞろえ */
  margin:0;
  padding:0;
}
.width1000{
  width: 1000px;
  text-align: center;     /* 中身を中央ぞろえ */
  vertical-align: middle; /* 垂直方向で中央揃え */
  border: 1px red solid;  /* 枠線を太さ1px、色を赤、普通の線 (確認用)*/
  margin: 0 auto;
}

ここでは、横並びのー例を紹介しただけですが、divタグなどにcssでレイアウトの設定をすることで、さまざまなレイアウトのページを作成することができます。「ボックスレイアウト　html」、「css デザイン」、「レスポンシブWebデザイン」などで検索すると、いろいろな解説記事や実例を見ることができますので、ぜひ、調べてみてください。

リンク (anchor)

htmlの特徴の一つであるリンク(アンカー:anchor)は、aタグを用います。具体的には下記のような書式です。 

<a href="https://www.shikoku-u.ac.jp"> 四国大学 </a>

この場合は、四国大学のページへのリンクを表しています。URLは絶対パスで表されています。絶対パスは、記載している場所に関わりなく、 示されているものです。すなわち、このページだけでなく、皆さんが作ったWebページでも全く同じ記載で同様にリンクできます。

絶対パスに対して相対パスもあります。相対パスの例は下記の通りです。 

<a href="AAA/BBB.html"> BBBへのリンク </a>

この場合、リンク元のファイルの場所と同じディレクトリ内にあるAAAというディレクトリの中にあるBBB.htmlへのリンクを表しています。

マウスカーソルが重なった時の表示切り替え

マススカーソルが重なった時に表示変わるボタンなどの処理はCSSだけでできます。例えば、aタグで表示されている文字の 背景を黄色にする場合、下記のようなコードをCSSに指定すると良いでしょう。 

 a:hover{ background-color: yellow;}

ボタンを押して、ダイアログを表示する

JavaScriptを用いることで、さまざまな処理を行うことができます。まずは、ボタンを押して処理を開始する仕組みを覚えましょう。 

<!doctype html>
<html>
<head>
    <meta charset="utf-8">
    <title>JavaScript </title> 
    <script>
        function show_message(){
            alert("押したね。");
        }
    </script>
</head>
<body>
    <h1>ボタンクリックでダイアログ</h1>
    <button onClick="show_message()">押してね</button>
</body>
</html>

まず、確認してほしい部分は、head部のscriptタグです。 このようにscriptタグで囲まれた部分にJavaScriptの プログラムをすることになります。このプログラムでは、 show_message()という関数（自作の命令文）を作成しています。 show_message()では、alert("...")という、ダイアログを表示する命令文が記載されているだけですが、ここを加筆修正すると、 さまざまな処理を行うことができます。

body内では、buttonタグの部分が新しい内容です。 onClick="..."で、ボタンをクリックした時の処理を指定しています。ここでは、head内で記述したshow_message()を呼び出すよう指定されています。

このプログラムでは、ボタンを押すと、「押したね」と書かれたダイアログが表示されます。このようにユーザの動作（イベント）によって動作することをイベントドリブンと呼び、多くのアプリケーションは、イベントドリブン型で開発されています。

DOM (Document Object Model)

htmlで記述された要素（h1やimgなど）の情報を取り出したり、書き換えたりするためにDOMを用います。実際に、先のプログラムを書き換えて確認してみましょう。

まず、alert行を下記のように書き換えます。

document.getElementById("title_text").innerHTML = "押したな〜";

次にh1タグにidを追加します。

<h1 id="title_text">ボタンクリックでここの文章が変わります。</h1>

どの要素かを指し示すために、idを設定していますが、"..."の部分は、同一ページで重複しない任意の文字列を書けば良いです。この場合、title_textとしました。JavaScriptで呼び出す時の指定方法は、document.getElementById("...")となります。そして、その要素の何を読み出す、あるいは、修正するかによって、その後ろの部分（今回の場合、innerHTML)は、変わります。

関数の引数

関数functionには、引数という処理に関わる値を取得する方法があります。下記のプログラムのように複数のボタンをそれぞれクリックした時の処理として、show_message()が呼び出されていますが、()の中に文字が指定されています。show_message内では、その内容はtxtとして扱われるように指定されています。すなわち、指定された文字が、h1の文章として表示されます。 

 <!doctype html>
<html>
<head>
<meta charset="utf-8">
<title>JavaScript </title> 
<script>
function show_message(txt){
    document.getElementById("title_text").innerHTML = txt;
}
</script>
</head>
<body>
<h1 id="title_text">ボタンクリックで文変更</h1>
<button onClick="show_message('1')">ABC</button>
<button onClick="show_message('2')">DEF</button>
<button onClick="show_message('3')">GHI</button>
</body>
</html>

条件分岐（if文）

Excelでも条件によって異なる処理ができましたが、ほとんどのプログラム言語でも、条件によって異なる処理が可能です。前回の引数の説明で用いたプログラムを改変し、引数の値によって処理を変えるプログラムを紹介します。 

 <!doctype html>
<html>
<head>
<meta charset="utf-8">
<title>JavaScript </title> 
<script>
function show_message(txt){
    if( txt == "1" ){
        document.getElementById("title_text").innerHTML = "ABCを押しましたね";
    }else if( txt == "2" ){
        document.getElementById("title_text").innerHTML = "DEFを押しましたね";
    }else if( txt == "3" ){
        document.getElementById("title_text").innerHTML = "GHIを押しましたね";
    }else{
        document.getElementById("title_text").innerHTML = "ERROR!!";
    }
}
</script>
</head>
<body>
<h1 id="title_text">ボタンクリックで文変更</h1>
<button onClick="show_message('1')">ABC</button>
<button onClick="show_message('2')">DEF</button>
<button onClick="show_message('3')">GHI</button>
</body>
</html>

まず、下記の部分に注目してください。

 if( txt == "1" ){ .... }

「txt == "1"」の部分は「txtの値が1と同じか？」ということを示しており、その条件を満たしている場合に、{}で囲まれた部分が実行されます。 今回は複数の条件を続けてチェックするしているので、 

 if( txt == "1" ){ 
     ..(1の時の処理).. 
 }else if( txt == "2" ){
     ..(1ではなく2の時の処理).. 
 }else if( txt == "3" ){
     ..(1でも、2でもなく3の時の処理).. 
 }else{
     ..(どの条件も満たさなかった時の処理).. 
 }

という構造になっています。この場合、txtが1であれば１の時の処理、そうではない場合、txtが2であるかチェックし、そうであれば、２の時の処理、そして、そうでない場合、txtが3であるかあるかチェックし、そうであれば、3の時の処理、そうでなければ、それ以外の場合の処理を行うというアルゴリズムになっています。

文字列と数値の変換

以前紹介した通り、文字列の連結も「+」を用います。さて、先のプログラムのalertの部分を下記のようにすると、どのように表示されるでしょうか？

alert("1+1="+1+1);

意図としては、「"」で挟まれた部分は文字列の連結で挟まれていない部分は数値として計算してもらいたい。 すなわち、「1+1=2」と表示してもらいたいのですが、実際には、後半の「1+1」も文字列として扱われ、「1+1=11」と表示されてしまいます。 これは、前半の扱いから、この式全体が文字列の連結として扱われてしまうためです。 他の言語では、文字列と数値（さらには、整数か、小数があるかなど)を明確に区別しますが、 JavaScriptでは、厳密には区別せず、その都度判断されます。 したがって、「1+1=11」のようなことが起こります。 この対策は色々あるのですが、明確にその文字や変数が、文字列あるいは数値であることを示す関数があります。 String()とNumber()です。文字通り、カッコの中をString(文字列）として扱うか、Number(数字)として、 扱うかを明確にするものです。今回の場合、下記のようにすると意図通りになります。 

alert("1+1="+Number(1+1));

記号 意味

> 左の値が大きい

< 左の値が小さい
<= 左の値が小さいか同じ

例えば、aが3で、「a==3」だと真（true)になり、aが3でなければ、疑(false)となります。

記号 意味

    for ( [初期化]; [継続条件]; [処理終了時の処理]  ){

}

    まとめると、まず、初期化し、継続条件を満たしていれば、繰り返したい処理を実行。そして、処理終了時の処理を行ったのち、継続条件のチェック。満たしれいれば、再度、繰り返したい処理を実施。。。。と継続条件を満たす限り繰り返します。

    var surname = ['阿部','加藤','佐藤','田中','中川','林','松本','矢部','和田'];
var point = [65, 48, 36, 80, 100, 94, 72, 67, 93];

    配列を紹介した時には、new Array();で宣言してから、値を代入していました。上記のように書くことで、配列として利用することと、値を代入することを一度に行うこともできます。

    このデータをtableで表示するプログラムを作成してみましょう。表示する部分の文字列の生成はfor文で作成すると以下のようになるでしょう。

    txt = "<table>";
for ( i = 0; i < surname.length; i++ ){
    txt = txt + "<tr><td>" + surname[i] + "</td><td>" + point[i]  + "</td></tr>";
}
txt = txt + "</table>";
